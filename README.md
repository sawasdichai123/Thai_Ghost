# Thai Ghosts — Spirits of Thailand 👻

An interactive, presentation-style website introducing famous ghosts of Thai
folklore to international audiences. English content with the Thai ghost name in
parentheses. Dark, blood-red, haunted theme with a slide-deck navigation flow.

## Spirits featured
1. Krasue (กระสือ) — The Floating Head
2. Phi Pop (ผีปอบ) — The Organ Eater
3. Phi Am (ผีอำ) — The Night Presser
4. Phi Nang Ram (ผีนางรำ) — The Dancing Spirit

## Run
Just open `index.html` in any browser — no build step required.

## Controls
- **← / → / Space** — navigate slides
- **F** — toggle fullscreen
- Click the dots at the bottom to jump to a slide
